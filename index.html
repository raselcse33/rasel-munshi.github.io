PHP Introduction

Scripting Language

স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে এক প্রকার প্রোগ্রামিং ল্যাঙ্গুয়েজে যেটা compilation ছাড়াই execute হয়।

example: php, javascript

Non-Scripting Languages

নন স্ক্রিপ্টিং ল্যাঙ্গুয়েজে হচ্ছে Compiled ল্যাঙ্গুয়েজে।

example: c, c++, java

Compiled

compiled হলো সেই জায়গা যেখানে আমার লেখা কোড টাকে machine language (binary code) convert করা হয়।

How Scripting Language Run

যেহেতু স্ক্রিপ্টিং ল্যাঙ্গুয়েজে compiled করার প্রয়োজন পরে না, তাই এটি রান করার জন্য ব্যবহার করা হয় the interpreter.

PHP Version Check

PHP ভার্সন চেক করার জন্য ব্যবহার করা হয়: phpversion()

Web Host

ওয়েব হোস্ট হলো একটা নির্ধারিত জায়গা যেখানে ওয়েবসাইট এর ফাইল রাখা হয়।

Example: Bluehost, HostGator, GoDaddy.

Web Server

ওয়েব সার্ভার হলো একটা টুল (tool) or software যেটা আমার ওয়েবসাইট রান করতে এবং ভিসিটরদের দেখতে সাহায্য করে।

Example: Apache or Nginx server.

XAMPP

XAMPP হলো একটা সফটওয়্যার প্যাকেজ যা আমার পিসি কে একটা লোকাল ওয়েব সার্ভার বানিয়ে ফেলতে পারে:

  • X: Cross-platform (works on Windows, Linux, and macOS)
  • A: Apache (a web server that runs your website)
  • M: MySQL (a database that stores your data)
  • P: PHP (a programming language used to build web applications)
  • P: Perl (another programming language, though less commonly used today)

Case Sensitivity

case sensitivity বলতে এইটা বুঝায় যে কোনো একটা লেটার বা ওয়ার্ড uppercase বা lowercase লিখলে এটি কি একই না ভিন্ন।

Example: Apple, apple

Variables

Variable

যেখানে value assign করা থাকে। var_dump() function এর মাধ্যমে আমরা variable টাইপ জানতে পারি.

Loosely Typed Language

PHP হল loosely typed language। এর মানে এখানে ভ্যারিয়েবল declured করার সময় বলার প্রয়োজন হয় না।

Example: $number = 5; $number = "PHP";

Types of Variable

  • String
  • Integer
  • Float (floating point numbers)
  • Boolean
  • Array
  • Object
  • NULL
  • Resource

Expressions

Expressions হলো value, variable, operator and function এর একটা combination যেটা একটা রেজাল্ট তৈরী করে।
